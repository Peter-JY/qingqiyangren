# 笃实书院

笃实书院以航天航空学院、工业工程系和软件学院为主要支撑院系（因此这三个院系没有本科生新生了），设置6个专业，六个专业不分流，仅通过选课区分，根据选课决定获得的专业学位。

> [清华大学笃实书院官网](https://www.dsc.tsinghua.edu.cn/)

### 专业体系

- 交叉工程
- 航空航天工程
- 工程力学
- 能源与动力工程
- 工业工程
- 软件工程
